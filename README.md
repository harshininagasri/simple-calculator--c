# Simple Calculator in C

## 📖 Description
A simple menu-driven calculator program written in C.  
It performs basic arithmetic operations and some advanced functions using the math library.

## ✨ Features
- Addition
- Subtraction
- Multiplication
- Division (with zero-check)
- Modulus (integer only)
- Power (x^y)
- Square Root

## 🛠️ How to Compile and Run
1. Save the code in a file named `calculator.c`.
2. Open terminal/command prompt and navigate to the folder containing the file.
3. Compile the program using:
   ```bash
   gcc calculator.c -o calculator -lm
